# 🗓️ Our Planner Premium

A premium, collaborative, mobile-first planner designed for local hosting on Termux via Tailscale.

## ✨ Features
- **Premium UI**: Glassmorphism design with Lucide icons.
- **Collaborative**: Shared server-side storage for Viki, Manja, and Admin.
- **Customizable**: Create custom categories with unique names and colors.
- **Termux Ready**: Optimized for Android server environments.
- **PWA**: Installable on mobile devices with offline support.

## 🚀 Quick Start (Termux)
1. Clone this repo into Termux.
2. Run `./start_termux.sh`.
3. Open on your device via Tailscale IP.

## 🛠️ Tech Stack
- **Frontend**: HTML5, Vanilla CSS (Premium Design), JS.
- **Backend**: Node.js + Express.
- **Icons**: Lucide Icons.
- **Fonts**: Outfit (Google Fonts).
